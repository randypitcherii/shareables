Naming Rule
target: ci_testing → default_schema used unmodified → CI injects DBT_DEFAULT_SCHEMA = dbt_<project>_pr<n>_build<run>  ·  Each run is a fresh, isolated workspace  ·  No cross-run contamination  ·  Schemas are disposable by design
